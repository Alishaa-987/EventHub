# 🎉 EventHub

EventHub is a modern event management platform that helps users discover, create, and manage events with ease. Whether it's conferences, workshops, meetups, or social gatherings, EventHub provides a seamless experience for both organizers and attendees.

<img width="852" height="900" alt="image" src="https://github.com/user-attachments/assets/f2e1e206-bd18-4f23-9da5-96b376cfc155" />

## ✨ Features

- 🔐 User Authentication & Authorization
- 📅 Create, Edit, and Delete Events
- 🎟️ Event Registration System
- 👥 Manage Attendees
- 🔍 Search and Filter Events
- 📍 Event Details & Location Information
- 📱 Responsive Design
- ⚡ Fast and User-Friendly Interface

## 🛠️ Tech Stack

### Frontend
- React.js / Next.js
- Tailwind CSS
- JavaScript / TypeScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT Authentication
- bcrypt Password Hashing

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18+ recommended)
- npm or yarn
- MongoDB

### Installation

1. Clone the repository

```bash
git clone https://github.com/Alishaa-987/EventHub.git
```

2. Navigate to the project directory

```bash
cd EventHub
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

5. Start the development server

```bash
npm run dev
```

## 🔄 Workflow

1. User registers or logs in.
2. Organizers create events.
3. Users browse available events.
4. Users register for events.
5. Organizers manage attendees and event details.
